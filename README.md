# TaskMate

TaskMate is a MERN-based platform that connects people who need help with tasks,
assignments, and projects with volunteers who can assist ethically and on time.

## Features
- Post tasks and assignments
- Volunteer-based task assistance
- Deadline-based accountability
- Clean UI with React
- Backend planned with Node, Express & MongoDB

## Tech Stack
- React (Vite)
- Node.js
- Express.js
- MongoDB (Atlas)
